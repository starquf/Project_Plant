# Project_Plant

## 1. 게임 설명
>여러 과일을 주어진 갯수만큼 자라게 하는 컨셉의 퍼즐 게임입니다.

1. 주변의 여러 타일들을 이용하여 밭에 물을 주어 과일을 자라게 할 수 있습니다.
2. 총 15개의 스테이지로 구성되어 있습니다.

***

## 2. 주요 코드 설명
### 2.1. 옵저버 패턴
옵저버 패턴을 적극 활용하여 타일 시스템과 게임의 클리어를 구현하였습니다.

### 2.2. 다양한 타일을 구현할 수 있는 객체지향적인 구조
다형성을 고려하여 효율적으로 다양한 타일을 구현할 수 있도록 구조를 설계했습니다.
